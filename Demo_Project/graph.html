Options

  • columns: Number of viewable thumbnails
  • imgHeight: jCarousel slider image height
  • imgWidth: jCarousel slider image width
  • jsonScript: JSON return file './scripts/json/json_results.js' anything that will return a JSON object can be used
  • ajaxData: Data array to pass to the AJAX call
  • jsonObject: JSON Object. Overrides jsonScript and ajaxData
  • cssClass: jCarousel css skin to use
  • imgEvent: 'click' or 'mouseenter'
  • afterEvent: Function to perform after a click/mouseover event occurs
  • animate: Turns off animation
  • jCarouselEnabled: Use jCarousel
